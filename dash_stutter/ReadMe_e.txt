========================================================================
    Universal Video Player : "Video Test Sample" 
========================================================================

--------------------------------------
DESCRIPTION

This is a sample web application that can be loaded by the Web Launcher.

How to customize
----------------

Sample Usage
------------
1. Copy the following files from the UVP library into the same directory as the sample
    files.
    - sce-uvp.esm.js
    - sce-uvp.css
    - sce-uvp-controls.css
    - hls.js
    - dash.js
2. Host the files on a local web server and run the Web Launcher targeting
    your host.
   	If you have Node/npm installed, you run 'npm install' and 'npm start' to
    perform this step.
